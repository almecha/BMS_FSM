<b>After cloning the repo:</b> <br>
Go to v2 directory 
```cd bms_fsm_v2```  <br>
then run 
```cmake -G "Unix Makefiles" -B Build -S .```  <br>
then go to newly created ```Build``` folder by running ```cd Build``` <br>
run ```make``` and after compilation run the executable by ```./C_StateMachineApp```  <br>
